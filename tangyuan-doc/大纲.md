## 4. 版本更新

+ 新增:INIX类型配置文件的支持；
+ 新增:占位替换变量配置文件的支持；
+ 新增:应用属性配置文件的支持；
+ 新增:异步任务线程池配置的支持；
+ 新增:<system-aop>标签，移除<init>标签；
+ 新增:服务统一返回XCO对象的控制参数；
+ 更新:TangYuanJPCBridge；
+ 修复:zongzi依赖；
+ 修复服务启动时的onlyProxyMode；
+ 修复默认值中负数的问题；
